# IOT-Experiments
Multiple minor experiment projects using Raspi-3&amp;4 [Python]

Executed a series of mini-projects focused on sensor integration and automation

Hands-on experiments exploring sor integration, automation, and loT systems:

•Setup & configuration of Raspberry Pi
•Controlled LED state via GPIO input commands
•Simulated traffic light system with timing logic
•Interfaced motor using ultrasonic sensor trigger
•Displayed temperature/​humidity via DHT11 sensor
•Activated buzzer in timed intervals.
•PIR motion detection with cloud upload to ThinkSpeak
•Bluetooth control for LED using smartphone input
•Installed and ran SQL queries on MySQL database
